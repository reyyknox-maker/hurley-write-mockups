## Key Principles from 21st.dev Components

✅ **Minimal:** No unnecessary decoration, every element serves a function
✅ **Professional:** Clean, readable, high contrast
✅ **Tech-Forward:** Modern design patterns, smooth animations
✅ **Subtle Interactions:** Smooth hover effects, not playful
✅ **High Contrast:** Dark text on light, light text on dark
✅ **Geometric:** Clean lines, shapes, grids
✅ **Asymmetric:** Not centered, modern layouts
✅ **Smooth Animations:** Transitions, reveals, subtle effects

## Specific Component Recommendations for Hurley Write

### Hero Section
- Use "Minimalist Hero" or "Modern Hero" as base
- Add "Beams Background" for subtle depth
- Keep text clean and readable
- Asymmetric layout (left text, right visual)

### Buttons
- Primary CTA: "Hover Button" + "Button with Icon"
- Secondary CTA: "Hover Button" with outline style
- Add smooth color transitions
- Include arrow icon for primary CTA

### Testimonials
- Use "Testimonial Cards" grid layout
- Apply "Minimal Testimonial" styling
- Add subtle hover effects (lift, shadow)
- Keep cards clean and minimal

### PROS Roadmap
- Use "Radial Orbital Timeline" pattern
- OR create 4-card grid with "Minimal Cards"
- Add subtle background (Beams or Grid)
- Color-code or accent the steps

### Backgrounds
- Hero: "Beams Background" or "Animated Gradient Background"
- Sections: "Background Paths" or "Grid Pattern"
- Keep subtle, not distracting
- Professional, tech-forward feel

### Typography
- Headlines: Bold, clean, high contrast
- Optional: "Typewriter Effect" or "Text Reveal" for animations
- Body: Regular weight, readable, high contrast
- Accents: Orange color for emphasis
