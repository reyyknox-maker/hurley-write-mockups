### 2. HERO SECTION (Above Fold)

**Left Side (Text):**
- **Label:** "For Technical Teams" (orange accent, small caps, subtle)

## Layout Principles

1. **Asymmetric Hero:** Left text, right visual. Breaks from centered templates.
2. **Generous Whitespace:** Premium feel comes from breathing room, not clutter. 2-4rem padding between sections.
3. **Clear Visual Hierarchy:** Hero → PROS → Testimonials → Results → CTA. Easy to scan.
4. **Card-Based Sections:** Modular, reusable components for consistency.
5. **Strategic Color Accents:** Orange used sparingly for maximum impact.
6. **Dark Sections for Depth:** Alternate between light and dark backgrounds to create visual interest.
7. **Gradient Backgrounds:** Use gradients (navy to orange) for premium feel and visual interest.

---

## Interactive Elements & Micro-Interactions

- **Hover States:** Buttons scale slightly, change color shade, shadow increases
- **Links:** Smooth underline animation on hover
- **Cards:** Subtle lift effect on hover (shadow increases)
- **Scroll Animations:** Sections fade in or slide up as user scrolls (optional, not required)
- **CTA Buttons:** Arrow icon animates on hover (slides right)
- **Smooth Transitions:** All animations should feel professional, not playful

---

## Responsive Design Breakpoints

- **Mobile (< 640px):** Single column, stacked sections, full-width buttons, adjusted font sizes
- **Tablet (640px - 1024px):** 2-column grids where appropriate, adjusted padding, responsive images
- **Desktop (> 1024px):** Full asymmetric layouts, multi-column grids, optimal spacing
